⚠ Disclaimer: This database is compiled from published agricultural / geological soil-survey literature (NBSS&LUP, TNAU, TN Geography records) and general IS-code (IS 1904, IS 6403) engineering practice. It is a broad, district-level starting point only. Soil type varies even within a single district. SBC values shown are indicative ranges, not test results. The GIS map is a stylized relative-position map, not a survey-grade boundary map. The AI Assistant is rule-based, not a trained model, and the document upload feature uses best-effort text/OCR extraction that should always be verified. A site-specific geotechnical investigation (trial pit / bore log / plate load test) is mandatory before finalising any real foundation design. This tool is for preliminary awareness and educational reference — not a substitute for a licensed geotechnical engineer's report.
